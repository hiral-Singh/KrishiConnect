# KrishiConnect

KrishiConnect is a full-stack MERN marketplace platform that connects farmers directly with consumers.

## Features
- User authentication
- Product listings
- Category filtering
- Farmer profiles
- Messaging system
- Order management

## Tech Stack
- React
- Redux Toolkit
- Node.js
- Express.js
- MongoDB Atlas

## Setup
1. Install dependencies
2. Configure environment variables
3. Run backend and frontend servers
